# ChIP-Seq data analysis pipeline for peak calling with MACS2

This repository contains the analysis scripts used to cross-reference TF binding data (NANOG and SOX2) with transcriptional bursting kinetics.

## Publication
G. G. Agsu et al., “Protein-protein interactions drive differences in the spatiotemporal dynamics of transcription factors NANOG and SOX2 in naïve pluripotent cells,” Dec. 2025, doi: https://doi.org/10.64898/2025.12.03.691924.

ChIP-seq Data: X. Chen et al., “Integration of external signaling pathways with the core transcriptional network in embryonic stem cells,” Cell, vol. 133, no. 6, pp. 1106–1117, Jun. 2008, doi: https://doi.org/10.1016/j.cell.2008.04.043.

Initial TF Targets: H. Xu et al., “ESCAPE: database for integrating high-content published data collected from human and mouse embryonic stem cells,” Database, vol. 2013, Jan. 2013, doi: https://doi.org/10.1093/database/bat045.

Noise Data: H. Ochiai et al., “Genome-wide kinetic properties of transcriptional bursting in mouse embryonic stem cells,” Science Advances, vol. 6, no. 25, Jun. 2020, doi: https://doi.org/10.1126/sciadv.aaz6699.
